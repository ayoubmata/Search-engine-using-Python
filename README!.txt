Afin d'utiliser le moteur de recherche sur votre machine, vous aurez besoin de :
    -MySQL (et MySQL workbench comme une interface de manipulationn et visualisation des tables) installer dans votre machine pour le stockage et la
     gestion des tables utilisées par le moteur de recherche. Les deuc peuvent etre télecharger et installer à travers l'outil
     MySQL installer que vous pouvez telecharger à partir de ce lien (https://dev.mysql.com/downloads/installer/).
    -Python 3.10 et un editeur de texte (Visual Studio code recommandé), ou bien un IDE de python comme pycharm 
    (vous pouvez le trouver dans le site de JetBrains et l'utiliser avec JetBrain free trial).
    -Installer Flask,Mysql et nltk à travers la commande "pip install" dans le terminal de l'IDE.
    -Prière de placer le dossier (docs) dans le chemin "C:\"
